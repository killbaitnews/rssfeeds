# KillBait News RSS Feeds

This repository hosts the RSS feeds for [KillBait News](https://killbaitnews.github.io/rssfeeds/). You can use these feeds to stay updated with the latest articles in different languages.

## Available RSS Feeds

- **Sitemap RSS**: Provides a complete sitemap of all RSS feeds.  
  [View Sitemap RSS](https://killbaitnews.github.io/rssfeeds/sitemap.rss)

- **English RSS Feed**: Get the latest news articles in English.  
  [View English RSS](https://killbaitnews.github.io/rssfeeds/en.rss)

- **Spanish RSS Feed**: Get the latest news articles in Spanish.  
  [View Spanish RSS](https://killbaitnews.github.io/rssfeeds/es.rss)

## Usage

You can subscribe to these RSS feeds using any RSS reader of your choice. Simply copy and paste the feed URL into your reader to start receiving updates.
